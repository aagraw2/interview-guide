# API and Database Design Interviews

This guide covers combined API and database design interviews where candidates are asked to design both the REST/GraphQL API and the underlying database schema for a given system.

## What to Expect

In an API and Database Design interview, you will:
1. Clarify functional and non-functional requirements
2. Design the REST API (endpoints, request/response, versioning, auth)
3. Design the database schema (entities, relationships, indexes)
4. Discuss trade-offs and scaling considerations
